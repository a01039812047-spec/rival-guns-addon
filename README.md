# Rival Guns Addon for Minecraft Bedrock

이 애드온은 Nerf Rival 시리즈를 모티브로 한 총기류와 아이템을 마인크래프트 베드락 에디션에 추가합니다.

## 포함된 아이템
- **라이벌 크로노스 XVIII-500**: 강력한 권총
- **라이벌 아폴로 XV-700**: 정밀한 소총
- **라이벌 네메시스 MXVII-10K**: 자동 사격 중화기
- **라이벌 라운드**: 모든 총기에 사용되는 노란색 구형 탄환
- **라이벌 마스크**: 보호용 전술 마스크

## 설치 방법
1. 이 저장소의 내용을 다운로드합니다.
2. `behavior_pack` 폴더를 마인크래프트의 `development_behavior_packs` 폴더에 넣습니다.
3. `resource_pack` 폴더를 마인크래프트의 `development_resource_packs` 폴더에 넣습니다.
4. 마인크래프트 월드 설정에서 실험적 기능(베타 API 등)을 활성화하고 팩을 적용합니다.

## 조합법
- **라이벌 라운드**: 철 조각 1 + 노란색 염료 1
- **총기류**: 철괴, 레드스톤, 노란색 염료를 조합하여 제작 가능합니다. 자세한 조합법은 게임 내 제작대에서 확인하세요.
